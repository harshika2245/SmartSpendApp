# Smart Spend and Budget Planner

Smart Spend and Budget Planner is a simple Java application designed to help users track their expenses and calculate their monthly spending. 
It allows users to manage their budget effectively by organizing expenses and providing basic spending summaries.

## Features

- Add and record daily or monthly expenses
- Calculate total monthly spending
- Basic budget management and planning
- User-input based console program
- Easy to understand and beginner-friendly Java code
